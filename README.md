# Portfolio personal - Salvador González

Portfolio web personal desarrollado con **HTML** y **CSS** para mostrar mis principales proyectos como desarrollador junior multiplataforma.

## Enlace al portfolio

[Ver portfolio online](https://salvadorgc1980.github.io/portfolio-salvador/)

## Proyectos incluidos

- **TextilTrack**: aplicación de escritorio en Java Swing y SQLite para la gestión de entregas textiles en una lavandería hospitalaria.
- **Área de Clientes Android**: prototipo de aplicación Android desarrollada con Kotlin, Jetpack Compose, MVVM y StateFlow.
- **Demo Metroidvania 2D**: demo técnica desarrollada en Unity y C# con exploración, UI, habilidades, enemigos e interacción con objetos.
